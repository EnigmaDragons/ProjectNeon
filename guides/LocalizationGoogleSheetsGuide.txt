Authorization:
- Click authorize on the GoogleSheetsService (this will take you to a page that it says untrusted)
- Click continue (this will then require you to approve the application)

To make changes:
- adjuect on google sheets the text for the category you want 
- find the same named object in localization folder on the unity project
- click pull

To Build:
- Open: Window > Asset Management > Addressables > Groups
- Click: Build > Clean > All
- Click: Build > New Build > Default Build Script
